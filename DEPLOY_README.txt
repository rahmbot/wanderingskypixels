HOW TO DEPLOY ON CLOUDFLARE PAGES:
  1. dash.cloudflare.com → Pages → wanderingskypixels → Create deployment
  2. Direct Upload → drag the entire wsp_site FOLDER
  3. Deploy — live in ~60 seconds

IMPORTANT: Upload the whole folder, not just index.html.
This is a multi-page site and all subdirectories are required.
